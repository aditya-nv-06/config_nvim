# Keybindings

This file lists the custom keybindings defined in this config.

## Core

| Mode | Key | Action |
|---|---|---|
| Normal | `;` | Enter command mode (`:`) |
| Insert | `jk` | Exit insert mode |

## File Explorer

| Mode | Key | Action |
|---|---|---|
| Normal | `<leader>e` | Toggle NvimTree |

## Telescope

| Mode | Key | Action |
|---|---|---|
| Normal | `<leader>ff` | Find files |
| Normal | `<leader>fa` | Find all files (including hidden + ignored) |
| Normal | `<leader>fg` | Live grep |
| Normal | `<leader>fb` | List open buffers |
| Normal | `<leader>fh` | Help tags |
| Normal | `<leader>fo` | Recent files |
| Normal | `<leader>fc` | Grep word under cursor |

## Buffer Navigation

| Mode | Key | Action |
|---|---|---|
| Normal | `<S-l>` | Next buffer |
| Normal | `<S-h>` | Previous buffer |
| Normal | `<leader>bd` | Close current buffer |

## Git / Tools

| Mode | Key | Action |
|---|---|---|
| Normal | `<leader>lg` | Open LazyGit |
